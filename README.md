# TripReady（トリップレディ）

渡航前の「モヤモヤ」を、国/都市を選ぶだけで1画面に整理するアプリ。  
初心者でも迷わない・一目で分かる・ワンタップで行動できるUIを目指す。

---

## 目的
海外旅行で毎回検索して詰まる情報を、国/都市ごとにまとめて提示する。  
「調べる→比較する→決める」の手間を最小化する。

---

## 対象ユーザー
- 旅行初心者〜中級者
- 出張・短期滞在の人
- 現地での移動/通信/支払いが不安な人

---

## 提供したい情報（完成版の構想）
※現在日付：2026-02-07（JST）

- 入国・滞在ルール：ビザ要否／滞在可能日数／入国拒否リスク（必要書類・残存期間など）※最終は一次情報で確認
- SIM・通信：現地SIMはどこで買う？価格とギガ／eSIMを事前に入れるべきか／空港・市内どっちが得か
- 配車・移動：現地で主流のタクシー/配車アプリ（安全・相場・空港→市内の最適ルート）
- デリバリー：UberEats相当の一般的アプリ（使い方・支払い・対応エリア）
- お金：通貨／日本円との体感レート（例：1万円≒◯◯）／現金・カード・QR決済の主流／ATM・両替の注意
- 観光地提案：おすすめ＋地図リンク＋住所の一覧化（移動順も組めると強い）
- 治安・危険回避：危険エリア／よくある詐欺・ぼったくり／夜の注意点（外務省情報が基準）
- 緊急時の連絡先：大使館/領事館、警察、病院、カード紛失、保険会社（ワンタップで出せると神）
- 保険・医療：医療費の高さ、旅行保険/医療保険/救急搬送の要否（渡航前に検討）
- 薬・持ち込み注意：持参薬の制限／現地で買える薬の注意（国で違う）※必要なら専門家に確認
- 電源・充電：コンセント形状／電圧／変換プラグ・変圧器の要否
- チップ・税・サービス料：チップ文化だけでなく、会計に上乗せされる費用の有無
- 言語テンプレ：配車・ホテル・デリバリーで使う定型文（住所の書き方、ピンの置き方）
- 祝日・営業時間：祝日で閉まる/混む、宗教イベントで営業時間が変わる（酒販売制限含む）

## MVP（まず作る最小）
完成版をいきなり作らず、まずは「国を選ぶ→5カード＋緊急ボタン」だけ作る。

### MVP仕様
- 国/都市を選択（検索 or ドロップダウン）
- 5つのカードを表示：
  1) 配車
  2) デリバリー
  3) 決済
  4) 翻訳
  5) 通貨（体感レート：例 1万円≒◯◯）
- 緊急ボタン：
  - 警察 / 救急 / 大使館（まずはリンク表示でOK、後で国別に最適化）

### MVPのデータ方針（最初）
- まずは手入力のサンプルデータ（例：タイ/バンコク、韓国/ソウルなど数都市）
- 後でAPIやスクレイピングではなく「公式一次情報リンク＋更新日」中心に拡張する

---

## 開発方針（初心者でも前に進むためのルール）
- まずはWebアプリ（スマホで使えるUI）
- 1画面完結、スクロールは短く
- 情報は「最短で行動できる」順に並べる
- 迷ったら：表示項目を減らす（増やすのは後）
- 最終確認が必要な項目は「公式リンク」へ誘導する
